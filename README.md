Занятие 6 Строки и массивы. Вариант 2. 
1) Написать программу, которая, используя метод бинарного поиска, выполняет
поиск в упорядоченном по возрастанию массиве. Массив предварительно
сортируется методом «пузырька».
2) Написать программу, которая проверяет, является ли введенная с клавиатуры
строка шестнадцатеричным числом.
